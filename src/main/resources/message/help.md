*Real finance help*
  
  `/amount [currency] [category]` - Save new expense
  
  Examples:
  `/14`
  `/999.90 RUB`
  `/0.99 usd`
  `/36.6 💊`
  `/10 usd 🍕`
  
  The default currency is Euro
  The default category is _Any_
  
  Possible categories:
  _Any:_ ☕️
  _House:_ 🏠
  _Food:_ 🍞🍕🍌
  _Health:_ 💊💉🌡
  _Sport:_ 🏊
  _Fun:_ 🎉
  _Travel:_ ✈️
  
  /total - Total chat expenses
  
  /clear - Clear chat expenses
  
  `/currency` - Change user default currency
  
  Examples:
  `/USD`
  `/rub`
